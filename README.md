# ComidaNoPrato

O ComidaNoPrato é o software que permite administrar o estoque de um ou vários restaurante(s) popular(es) através do cadastro de entrada e saída de alimentos e bebidas. O sistema controla o estoque de alimentos, validade, estima projeções mensais a partir do estoque, quantidades diárias de refeições e balanceamento de estoque entre restaurantes. Além disso, há o cadastro de agricultores sustentáveis, permitindo ao Gerente Geral ter melhores opções de onde comprar alimentos para o(s) restaurante(s).
